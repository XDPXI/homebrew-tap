# Install Tap

```bash
brew tap XDPXI/tap
```

# Apps

- [Mono](https://xdpxi.itch.io/mono)
  ```bash
  brew install --cask XDPXI/tap/mono
  ```
- [XD's Code](https://github.com/XDPXI/XDs-Code)
  ```bash
  brew install --cask XDPXI/tap/xds-code
  ```
- Darkwaters
  ```bash

  ```
- [Sponge](https://sponge.xdpxi.dev)
