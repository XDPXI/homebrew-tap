# Install Tap

```bash
brew tap XDPXI/tap
```

# Apps

- Mono
  ```bash
  brew install --cask XDPXI/tap/mono
  ```
- XD's Code
  ```bash
  brew install --cask XDPXI/tap/xds-code
  ```
- Darkwaters
  ```bash
  brew install --cask XDPXI/tap/darkwaters
  ```
- Sponge
  ```bash
  brew install XDPXI/tap/sponge
  ```
- Gin
  ```bash
  brew install --cask XDPXI/tap/gin
  ```
- GinCMD
  ```bash
  brew install XDPXI/tap/gincmd
  ```
